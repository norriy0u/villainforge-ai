# 👿 Villain Origin — Forge Your Path

> **VishwaNova 2026 · National Level Weboreel AI Hackathon**

Every villain has a reason. What's yours? A highly atmospheric procedural origin story generator that takes your inputs and creates a cinematic, fully immersive dossier of your ascension to darkness.

## ✨ Features

- 🌩️ **Immersive Entry Hook**: A true-black entrance screen featuring CSS-drawn red mist silhouettes and a multi-point polygon `clip-path` animation generating a flickering blood-red lightning storm in the background. Continuous CSS ember particles float upward across the viewport.
- 📁 **Classified Dossier UI**: Takes inputs using a dark-mode military dossier aesthetic complete with a monospaced `<input>` layout and staggered sliding entrance animations.
- 💥 **Shatter Transition**: Submitting the form triggers a dramatic CSS `clip-path` "glass shatter" effect that envelopes the screen to reveal the loading void.
- 📖 **Procedural Narrative Generation**: Takes the user's name, betrayal, power, and weakness, and algorithmically writes a dramatic, 3-paragraph villain origin story.
- 🎭 **Cinematic Reveal**: The generated villain title animates onto the screen letter-by-letter with a heavy drop-shadow filter. Below it, an AI-generated Pollinations.ai dark fantasy portrait slowly transitions from pure grayscale into color via a 3-second CSS filter animation.
- 🎚️ **Threat Meter**: The final dossier calculates your randomized "Threat Level" (Alpha, Demon, or Omega class) and visually drives a CSS gradient progress meter to that level.

## 🛠️ Tech Stack

- **HTML5** — Multi-screen SPA structure and semantic layout.
- **Vanilla CSS3** — Intense `clip-path` animations, `mix-blend-mode` effects, advanced text-shadow rendering, radial mist gradients.
- **Vanilla JavaScript** — Real-time narrative interpolation, sequential timeline animation delays.
- **Web Audio API** — Procedural heavy bass drone stings and low-frequency impact hits.
- **Pollinations.ai** — Dynamic portrait rendering based on generated origin themes.

## 📸 Try It Out

Simply double-click the `index.html` file to open it in any modern browser. No servers, build steps, or dependencies required.
